-------------- Command Documentation --------------

Document only:

#section Kapitel                start a new section

#subsection Subkapitel          start a new subsection

#newpage

#newline
----------

Beamer only:

#title                          write frame title

#subtitle                       write frame subtitle

#pause                          set page clone entry

#frame / frame#                 start / end a slide page

----------

No specific documentclass:

#doctitle                       sets the documents title

#author                         set the documents author

#fat text fat#                  write text fat

#ital text ital#                write text italic
                                                _________
#table <columns>                Create a Table |lala|hihi|, <columns> must be 2
    lala|hihi
table#

#newline                        new line

#todo text todo#                creates a todo annotation

#img small bildname             include image, default: medium
#img medium bildname
#img bildname
#img big bildname

----------

Autoreplacements:

&                               get replaced through \&

#                               get replaced through \#

...                             get replaced through \ldots

<, >                            get replaced through $<$, $>$
