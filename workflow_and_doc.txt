-------------- Command Documentation --------------

Document only:

#section Kapitel                start a new section         takes whole line

#subsection Subkapitel          start a new subsection      takes whole line

#newpage                                                    takes whole line

#newline                                                    takes whole line
----------

Beamer only:

#title                          write frame title           takes whole line

#subtitle                       write frame subtitle        takes whole line

#pause                          set page clone entry        takes whole line

#frame / frame#                 start / end a slide page    takes whole line

----------

No specific documentclass:

#doctitle                       sets the documents title    takes whole line

#author                         set the documents author    takes whole line

#fat text fat#                  write text fat

#ital text ital#                write text italic
                                                _________
#table <columns>                Create a Table |lala|hihi|, takes whole line
    lala|hihi                   <columns> must be 2
table#

    - Blabla                

#titlepage                      creates a title/titlepage   takes whole line
                                (article/report)

#newline                        new line                    takes whole line

#todo text todo#                creates a todo annotation

#img small bildname             include image,              takes whole line
#img medium bildname            default: medium 
#img bildname
#img big bildname

----------

Autoreplacements:

&                               get replaced through \&

#                               get replaced through \#

...                             get replaced through \ldots

<, >                            get replaced through $<$, $>$
